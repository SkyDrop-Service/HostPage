Development

Getting Started

SkyDrop is Open-Source for the purpose of allowing any and all people to build on it.

Contributing to the Repository

Make changes to the server code? Create a pull request and we will review as soon as we can get to it!

Make a new method for calling the API? Create an Issue on the GitHub page, and we will verify it and add it to the list of options for users as soon as we can!

Join The Team

We develop SkyDrop in our free time. If you would like to join the team to help approve requests and help develop, you can email us at admin@skydrop.space
